I am located in Sydney, Australia.

Javascript itself, has an interesting history. It was originally conceived in 1995 as a lightweight language to complement Java, with the requirements that it would appeal to non-professional programmers and be interpretable for easy embedding in webpages. Brendan Eich, it's inventor, famously created a working prototype in ten days.

It was not taken seriously until 2005 when the introduction of AJAX led to it becoming an essential part of application development and moved it into mainstream development. It's position was further strengthened in 2009 with the widespread adoption of Single Page Applications and by Server Side deployment led by the AngularJS framework, and by NodeJS respectively.

Douglas Crockford summed up the good parts of Javascript in a few lines, namely lambda functions, functions as first-class objects, loose typing, dynamic objects, and an expressive object literal notation. In fact, javascript was the first lambda language to go mainstream. It is these good parts that I find so appealing, hence this blog.
