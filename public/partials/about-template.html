ABOUT

Last Updated on 11th March 2018

Front End Engineer is a blog template created by Snapdragonxc, a software developer. The client-side code was built with AngularJS and UI-Router. The server-side code uses MongoDB and ExpressJS. Authentication is with Postman. The site is hosted on an Amazon, AWS, EC2 Ubuntu instance with an Nginx server.

The front-page styling and design, with masonry image packing and upside-down navigation, was based on the www.wix.com template: graphic-design-portf. The remainder of the styling is standard Bootstrap, with flat-buttons and controls.

The blog template is currently populated with blogs created from the 1998 Netscape manual, 'Core Javascript Guide 1.4', which is available for download as a PDF.

Code syntax highlighting is with the PrismJS library using a default colour scheme.
